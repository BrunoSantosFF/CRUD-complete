Criando os arquivos para o projeto: 
  npx create-react-app my_crud 
  cd my_crud

Foi criada a pasta backend (javascript) e foi feita a instalação das extensões:
  mkdir backend
  cd backend 
  npm init -y 
  npm install express mysql cors


Para executar o projeto:
  npm start
